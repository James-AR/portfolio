Deployed Site: Redux/Django Lead Manager

Full stack application completed with React Redux for front end state management, and Django for knox REST framework, authentication, authorization, and database. Also with added feature of a custom error action/reducer to display errors with React-Alert package. A user can register for a new account, or log in with existing account, and create a list of lead managers complete with name, email, and message. Authentication is done with token authentication and local storage.

Github: https://github.com/James-AR/react_django
